# 第5章 助動詞

## 概要

- **目的**: 助動詞の形態的特徴と意味の幅を整理し、文脈依存のニュアンスを掴ませる。
- **学習の軸**: 後ろは動詞原形、否定・疑問での語順、過去形との意味差を体系化する。

---

## A 助動詞の種類と性質

- **形態のポイント**: 主語で形が変わらない／後ろは動詞原形／否定・疑問で助動詞を前に出す。
- 時制を表すときは **have + p.p.** を組み合わせて扱う。

## B can / may / must

| 助動詞 | 主な意味 | 注意点 |
| --- | --- | --- |
| **can** | 能力・許可・可能性 | 事実的可能性を表す *be able to* との対比。 |
| **may** | 許可・推量 | 許可と推量で語調が変わる。 |
| **must** | 義務・強い推量 | **must not ≠ don't have to** を徹底。 |

## C would / should

- **would**: will/shall の過去形（丁寧依頼・過去の習慣）+ 仮定法での用法。
- **should**: 義務・当然・助言・仮定法過去完了での帰結部など、多義性を整理。

## D その他の助動詞

- **ought to, used to, dare**（助動詞用法）など、形と意味のセットをまとめる。

## E 助動詞 + have + p.p.

- **現在からの過去推量** を表す形。現在形とのニュアンス差を対照表で示す。

## F 助動詞を含む慣用表現

- **had better, would rather, cannot help ...ing, have only to** など、定型として覚える表現を一覧化。
