# 第8章 動名詞

## 概要

- **目的**: 動名詞が持つ名詞的性質と動詞的性質の両面を理解し、不定詞との使い分けを整理する。
- **学習の軸**: S/O/C/M どの位置でも働くこと、目的語としての動名詞をとる動詞群の把握。

---

## A 動名詞の役割

- **形**: **動詞 + -ing** が名詞化。
- **機能**: S / O / C / M を自由に担う。
- **所有格/目的格の関係**: **my going / me going** の使い分けに触れる。

## B 不定詞との使い分け

- 意味差をもつ動詞群を一覧化し、頻出例を反復練習。
  - **to 不定詞のみ**: want, hope, decide など。
  - **動名詞のみ**: enjoy, avoid, admit, mind など。
  - **意味差が出る**: remember, forget, try, stop など。

## C 使役・知覚動詞との関係

- 第5文型の復習として、**O + 動名詞** との意味差（継続中の動作/一部を見た等）を確認する。
