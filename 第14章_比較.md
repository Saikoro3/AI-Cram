# 第14章 比較

## 概要

- **目的**: 原級・比較級・最上級の構造と意味を整理し、不規則変化や慣用比較表現を定着させる。
- **学習の軸**: than/ as の後ろの語順、倍数表現、最上級の修飾範囲を明確にする。

---

## A 原級・比較級・最上級

| 段階 | 形 | ポイント |
| --- | --- | --- |
| **原級** | as + 形/副 + as | 同等比較。否定で「〜ほど…ない」。 |
| **比較級** | 形/副 + **-er** / more + 形/副 + **than** | than の後ろは節でも可。 |
| **最上級** | the + **-est** / the most + 形/副 | **the** の省略不可。of/in で範囲指定。 |

- 不規則変化（good-better-best, many-more-most など）を一覧化。

## B 倍数表現と比較慣用句

- **倍数 + as + 形/副 + as**, **倍数 + the + 比較級 + of** などを使い分け。
- **less than / no less than / not more than / no more than** の意味差を整理。
- **the 比較級 ..., the 比較級 ...** の相関比較を例文で練習。

## C 形容詞・副詞の級の選択

- 音節数による **-er/-est と more/most** の選択基準を表で示す。
- 修飾語（much/far/a little など）で比較の度合いを調整する。
