## 第15章 仮定法

第15章 仮定法

仮定法は、「事実に反すること」や「実現可能性の低いこと」を述べるための「法（Mood）」の一種である。核心は、動詞の時制を意図的に「過去」にずらす（バックシフトする）ことで、現実との距離（＝非現実性）を表現する点にある。
さらに、大前提として、仮定法の最低構成条件は、「過去形の助動詞」であることを強調すべきである。過去形の助動詞の存在がトリガーとなることで、多様なパターンに対応でき、仮定法を予測できる。


### A 仮定法過去

概念: 「現在の事実に反する仮定」13。
形式: $If + S' + \text{過去形}..., S + would/could/might + \text{原形}...$
指導の核心: $If$ 節の動詞が「過去形」 13 であるにもかかわらず、意味内容は「現在」の仮定である、という「時制のズレ」を理解させる。$be$ 動詞は $were$ を用いるのが原則（口語では $was$ も可）。

### B 仮定法過去完了

概念: 「過去の事実に反する仮定」13。
形式: $If + S' + \text{過去完了形}(had + pp)..., S + would/could/might + have + pp...$
指導の核心: $If$ 節が「過去完了形」 13 で、意味内容は「過去」の仮定であること。

### C if節中の were to, should

未来に対する仮定。
$were to...$: 実現可能性が極めて低い、あるいは不可能な未来の仮定（$If I *were to* be born again...$）。
$should...$: 「万一～ならば」（実現可能性は低いが、ゼロではない）。

### D 倒置による if の省略

仮定法において $If$ が省略されると、続く $If$ 節内で主語と（助）動詞の倒置（疑問文の語順）が発生する 14。
$If I *were* you,...$ → $*Were* I you,...$
$If I *had* known...,...$ → $*Had* I known...,...$
$If you *should* need...,...$ → $*Should* you need...,...$

### E 仮定の意味を含む if 節以外の表現

$If$ 節を用いずに仮定法（帰結節の $would/could + V$）が使われる表現。
前置詞句: $Without...$ / $But for...$ （～がなければ、～がなかったら）
主語: $A careful driver would not do such a thing.$ （注意深い運転手なら...しないだろう）
不定詞: $To hear him talk, you would...$ （彼が話すのを聞けば...）

### F 仮定法を含む慣用表現
仮定法の「時制のバックシフト」ルールは、$If$ 節以外にも一貫して適用される。

仮定法の「時制のバックシフト」ルールは、$If$ 節以外にも一貫して適用される。
$I wish...$: （～であればいいのに／～であればよかったのに） 13
$I wish + \text{仮定法過去}$: 「現在」の事実に反する願望 13。
（例：$I wish I *were* rich.$）
$I wish + \text{仮定法過去完了}$: 「過去」の事実に反する願望・後悔 13。
（例：$I wish I *had bought* it.$）
$as if / as though...$: （まるで～のように） 16
指導の核心は、$as if$ 節内の時制が、主節の時制との「相対関係」で決まる点にある 17。
$as if + \text{仮定法過去}$: 主節と「同じ時」の仮定。
（例：$He talks as if he *knew* everything.$ ＝ 現在、知っているかのように話す）
$as if + \text{仮定法過去完了}$: 主節より「前」の時の仮定 17。
（例：$He talks as if he *had known* everything.$ ＝ 現在、(過去に)知っていたかのように話す）
$It's (high/about) time...$: $It's time + S + \text{過去形}$ （そろそろ～すべき時間だ）。
仮定法は、A, B, F の各項目が「時制をずらして非現実性を示す」という一つの統一ルールで支配されていることを理解させることが重要である。

仮定する内容の時制 | 表現（If / I wish / as if） | 使用する動詞の形
---|---|---
現在 の事実に反する | $If...$ / $I wish...$ 13 / $He talks as if...$ | 仮定法過去（過去形 / $were$） 17
過去 の事実に反する | $If...$ / $I wish...$ 13 / $He talked as if...$ | 仮定法過去完了（$had + pp$） 17


### G 仮定法現在

$suggest, demand, require, propose, recommend$ などの「要求・提案・命令」を表す動詞に続く $that$ 節内で、動詞が原形（または $should + \text{原形}$）になる用法。
例：$He suggested that she *(should) go* at once.$
