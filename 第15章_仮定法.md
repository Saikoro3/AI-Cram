# 第15章 仮定法

## 概要

- **目的**: 事実と異なる仮定を表すときの時制のずらし方と、if 以外の仮定表現を整理する。
- **学習の軸**: 仮定法過去・仮定法過去完了の形、倒置や混合文の扱い。

---

## A 仮定法過去・仮定法過去完了

| 種類 | 形 | ニュアンス |
| --- | --- | --- |
| **仮定法過去** | If + S + **過去形**, S + **助動詞過去 + 原形** | 現在の事実と逆 | 
| **仮定法過去完了** | If + S + **had + p.p.**, S + **助動詞過去 + have + p.p.** | 過去の事実と逆 |

- **be動詞は were** が原則である点を強調。

## B if 以外の表現

- **should + S + V**, **were + S + to V**, **had + S + p.p.** などの倒置で条件節を導く。
- **without / but for / otherwise** を用いた仮定表現。

## C 混合仮定法

- 条件節と帰結節の時制が異なるパターン（If S had p.p., S would V now.）を扱う。
- 時制の基準時を図で整理し、誤用を防ぐ。
