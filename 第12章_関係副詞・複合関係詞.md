# 第12章 関係副詞・複合関係詞

## 概要

- **目的**: 関係副詞の機能と、複合関係詞の範囲を整理し、場所・時・理由・方法を示す関係節を扱えるようにする。
- **学習の軸**: 前置詞+which との書き換え、強調構文との関係、複合関係詞の訳し分け。

---

## A 関係副詞の基本

| 関係副詞 | 置き換え | 例 | 役割 |
| --- | --- | --- | --- |
| **when** | at/on/in + which | the day **when** ... | 時 |
| **where** | in/on/at + which | the place **where** ... | 場所 |
| **why** | for which | the reason **why** ... | 理由 |
| **how** | in which / the way | the way **how/in which** ... | 方法 |

- **前置詞+which** で書き換え可能なことを示す。

## B 強調構文との関係

- It is/was + 強調したい部分 + **that** + 残り で、関係詞的な節を含む場合の語順を確認する。
- 関係副詞が **that** に置き換わる場面（the way that ... など）を整理。

## C 複合関係詞

- **whoever/whichever/whatever** = any + 関係詞。
- **複合関係副詞**: whenever/wherever/however など。
- 「〜する人は誰でも」「どんな〜でも」などの幅広い意味を示し、名詞節/副詞節の両方で使われることを確認。
