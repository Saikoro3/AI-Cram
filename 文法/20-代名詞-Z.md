# 第20章 代名詞

前章（第19章）では「名詞」について学びました。
本章では、その名詞の繰り返しを避け、文をスマートにする**代名詞 (Pronouns)** を学習します。
特に「不定代名詞」は、入試や検定試験で頻出の紛らわしい項目です。図形的なイメージを持って整理しましょう。

---

## 1. 人称代名詞・再帰代名詞・it

### A. 人称代名詞の格変化

代名詞は、文中での役割（主語・所有・目的語）によって形を変えます。これを「格」と呼びます。

| 格 | 主格 (〜は/が) | 所有格 (〜の) | 目的格 (〜を/に) | 所有代名詞 (〜のもの) |
| :--- | :--- | :--- | :--- | :--- |
| **1人称** | **I** | **my** | **me** | **mine** |
| **2人称** | **you** | **your** | **you** | **yours** |
| **3人称(男)** | **he** | **his** | **him** | **his** |
| **3人称(女)** | **she** | **her** | **her** | **hers** |
| **3人称(物)** | **it** | **its** | **it** | (なし) |
| **複数** | **we / they** | **our / their** | **us / them** | **ours / theirs** |

> [!NOTE]
> **所有代名詞** (`mine`, `yours` など) は、「所有格 + 名詞」の働きをします。
> *   This is **my pen**. = This is **mine**.

### B. 再帰代名詞 (...self)

「〜自身」を表す `...self` (複数は `...selves`) には、大きく2つの用法があります。

#### 1. 再帰用法
主語が行った動作が、**再び主語自身に戻ってくる** 場合に使います（S = O の関係）。

*   **He killed himself.**
    （彼は自殺した。/ 彼は自分自身を殺した。）
    *   比較: *He killed him.* だと、「彼は（別の）彼を殺した」という意味になります。

#### 2. 強調用法
「他でもない、その人自身が」と強調するために添える用法です。省略しても文は成立します。

*   **I myself** did it. / I did it **myself**.
    （**私自身が**それをやったのだ。）

> [!TIP]
> **慣用表現**
> *   **by oneself**: 一人で (= alone)
> *   **for oneself**: 独力で (= without help)
> *   **beside oneself**: 我を忘れて（自分自身の横に魂が抜け出ているイメージ）

### C. it の特別用法

`it` は単に「それ」と訳すだけでなく、漠然とした状況や環境を指す場合があります。

1.  **天候・時間・距離・明暗**
    *   **It** is raining. （雨が降っている。）
    *   How far is **it** to the station? （駅までどのくらいですか？）
2.  **状況の it**
    *   Take **it** easy. （気楽にいこうよ。）
    *   **It** seems that he is sick. （彼は病気のようだ。）
3.  **形式主語・形式目的語**（第7章・第13章復習）
    *   **It** is important **to study English**. （英語を学ぶことは重要だ。）

## 2. 指示代名詞・疑問代名詞

### A. 指示代名詞 (this / that)

「これ」「あれ」と指し示す代名詞です。

| 近称 (これ) | 遠称 (あれ) |
| :--- | :--- |
| **this** | **that** |
| **these** (複数) | **those** (複数) |

#### 重要構文: that of ... / those of ...
比較の文などで、**既に出た名詞の繰り返しを避ける**ために `that` (単数) や `those` (複数) を使います。

> [!IMPORTANT]
> **The climate of Japan is milder than that of England.**
> （日本の気候は、イギリスの**それ（=気候）** より穏やかだ。）
>
> *   × ... than England. とすると、「日本の気候」と「イギリスという国」を比較することになり不自然です。
> *   名詞が複数形なら `those of` を使います。

### B. 疑問代名詞

「誰」「何」「どれ」を尋ねる代名詞です。文中で S, O, C の役割を果たします。

*   **Who** broke the window? （誰が窓を割ったの？ / Sの役割）
*   **What** are you doing? （何をしているの？ / Oの役割）
*   **Which** is yours? （どっちがあなたの？ / Sの役割）

## 3. 不定代名詞

特定の人や物を指さず、漠然と「ある人」「ある物」を指す代名詞です。

### A. one と it の違い

*   **it**: **「まさにそれ」**（特定の既出名詞）
    *   I bought a pen. I lost **it**. （ペンを買った。**そのペンを**なくした。）
*   **one**: **「同種類のどれか1つ」**（不特定の可算名詞）
    *   I lost my pen. I must buy **one**. （ペンをなくした。**（代わりの）1本を**買わなきゃ。）

### B. another / the other / others

「別のもの」「残りのもの」を表す表現は、**「特定できるか（theがつくか）」** と **「単数か複数か」** で区別します。

| 表現 | イメージ・意味 |
| :--- | :--- |
| **one ... the other** | **2つあるうちの**、「1つは...、**残りの1つは**...」。<br>（残り1つと特定できるので `the` がつく） |
| **one ... another** | 3つ以上あるうちの、「1つは...、**別のもう1つは**...」。<br>（`an` + `other` = 特定できない別の1つ） |
| **some ... others** | たくさんあるうちの、「〜なものもあれば、**（別の）〜なものもある**」。<br>（漠然としたグループなので `the` はつかない） |
| **the others** | **「残りのすべて」**。<br>（残り全部と特定できるので `the` + 複数 `s`） |

> [!WARNING]
> **解説**:
> *   箱の中にボールが2個 → 1個取ったら残りは自動的に決まる → **the other**
> *   箱の中にボールが100個 → 1個取って、もう1個取っても、どれか特定できない → **another**

### C. each / every

*   **each**: 「それぞれ」（代名詞・形容詞の両方あり）
    *   **Each** of us has a dream. （私たちのそれぞれが夢を持っている。）
*   **every**: 「すべての」（**形容詞のみ**）
    *   × Every of us... とは言えません。**Every student**... のように必ず名詞を伴います。

### D. both / either / neither （2つのもの）

2つのものについて話すときに使います。

| 表現 | 意味 | 扱い |
| :--- | :--- | :--- |
| **both** | **両方とも** | 常に複数扱い |
| **either** | **（2つのうち）どちらか** | 単数扱い |
| **neither** | **（2つのうち）どちらも〜ない** | 単数扱い |

*   **Neither** of the answers **is** correct. （答えはどちらも正しくない。）

### E. all / none （3つ以上のもの）

3つ以上のものについて話すときに使います。

| 表現 | 意味 | 反意語 |
| :--- | :--- | :--- |
| **all** | **すべて** | **none** (どれも〜ない) |

*   **None** of the students knew the answer. （生徒の誰も答えを知らなかった。）

### F. some / any

一般的に、肯定文では `some`、否定文・疑問文では `any` を使います。

*   I have **some** money. （いくらかお金を持っている。）
*   I don't have **any** money. （お金を少しも持っていない。）
*   Do you have **any** money? （お金を持っていますか？）

> [!NOTE]
> **例外**:
> *   **勧誘・依頼**の疑問文では `some` を使います（Yesの答えを期待しているため）。
>     *   Would you like **some** coffee? （コーヒーはいかがですか？）
> *   **「どんな〜でも」** という意味の肯定文では `any` を使います。
>     *   **Any** child can do it. （どんな子供でもそれができる。）
