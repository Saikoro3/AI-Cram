# 第5章 助動詞

> **学習のポイント**
> 助動詞は、動詞に**「話し手の気持ち（判断・態度）」**をプラスする言葉です。
> 単なる動作の事実（「走る」「食べる」）に、「～できる」「～しなければならない」「～かもしれない」といった**彩り（Mood）**を加えるのが助動詞の役割です。

---

## 1. 助動詞の基本ルール

まず、すべての助動詞に共通する「鉄の掟」を確認しましょう。

| ルール | 説明 | 例 |
| :--- | :--- | :--- |
| **① 形が変わらない** | 主語が3人称単数でも `s` はつきません。 | He **can** swim. (× cans) |
| **② 後ろは原形** | 助動詞の後ろは必ず**動詞の原形**です。 | You must **go**. (× goes / went) |
| **③ 否定文** | 助動詞の直後に `not` を置きます。 | I **will not** (won't) go. |
| **④ 疑問文** | 助動詞を主語の前に出します。 | **Can** you hear me? |

---

## 2. 基本の助動詞 (can, may, must)

これらは「1つの単語で複数の意味」を持つため、文脈判断が重要です。

### A. can（能力・許可・可能性）

1. **能力**：「～できる」
   - $= \text{be able to}$
   - **重要:** 過去の「具体的な成功体験（～できた）」を表す場合は、`could` ではなく `was/were able to` を使うのが原則です。
2. **許可**：「～してもよい」
   - `May` よりもカジュアルな響きがあります。
3. **可能性**：「～でありうる／～しうる」
   - 「理論上ありえる」というニュアンスです。
   - 否定の `cannot` は「～のはずがない」という**強い否定推量**になります（後述）。

### B. may（許可・推量）

1. **許可**：「～してもよい」
   - 上下関係がある場面や、改まった場面で使われます。
2. **推量**：「～かもしれない」
   - 確信度は50%程度。「そうであるかもしれないし、ないかもしれない」。

### C. must（義務・強い推量）

1. **義務**：「～しなければならない」
   - $= \text{have to}$
   - **超重要:** 否定形の意味の違いに注意！

| 形 | 意味 | ニュアンス |
| :--- | :--- | :--- |
| **must not** | **～してはいけない** | **禁止**（強い命令） |
| **don't have to** | **～する必要はない** | **不必要**（自由） |

> **解説**
> `must` は「逃げ道なし」の圧力です。だから `must not` は「逃げ道なしでダメ＝禁止」。
> 一方、`have to` は「客観的な義務」です。`don't have to` は「その義務はない＝しなくていい（してもいい）」となります。

2. **強い推量**：「～にちがいない」
   - 確信度は90%以上。根拠がある確信です。
   - 対義語は `cannot`（～のはずがない）です。`must not`（禁止）ではないので注意しましょう。

## 3. will / would / should

### A. will の否定文
**will** は第3章で学んだように、「意志」を表す助動詞です。
その否定形 **won't**（will not）は、「意志がない」という意味から転じて、**「絶対に～しない／どうしても～しない」**という強い拒絶や、物事が全く起こらないことを表します。

> **The door won't open!**  
> （ドアが全く開かない！）
> 
> *ドアに「開く意志がない」→ 頑として開かない、というニュアンス。*

> **He won't listen to me.**  
> （彼は私の話を絶対に聞こうとしない。）

このように、**will** が「意志」を表すことを理解すると、否定文の「意志がない」→「どうしても～しない」という意味の成り立ちが見えてきます。

### B. would の用法
`will` の過去形としての用法の他に、以下の独自用法があります。

1. **過去の習慣**：「（昔は）よく～したものだ」
   - $He \textbf{would} often go fishing.$
   - ※「昔の状態」は表せません（それは `used to` の役割）。
2. **丁寧な依頼**：「～していただけませんか？」
   - $ \textbf{Would} you open the window? $
3. **仮定法**（第15章で詳しく扱います）

### C. should の用法
1. **義務・当然**：「～すべきだ／～するはずだ」
   - `must` ほど強制的ではなく、「常識的に考えてそうするよね」というニュアンス。
   - $= \text{ought to}$
2. **推量**：「（きっと）～のはずだ」
   - 「当然そうなるはず」という予測。

## 4. その他の助動詞 (used to / ought to)

### A. used to
「以前は～だった（が、今は違う）」という**「現在との対比」**が核心です。

| 用法 | 意味 | would との違い |
| :--- | :--- | :--- |
| **① 過去の習慣** | （以前は）よく～した | `would` とほぼ同じだが、「今はしていない」含意が強い。 |
| **② 過去の状態** | （以前は）～だった | **`would` は不可！** <br> (例) There **used to** be a park here. |

### B. ought to
- 意味は `should` とほぼ同じ（～すべきだ）。
- 否定形の位置に注意： `ought \textbf{not} to V`

---

## 5. 〈助動詞 + have + 過去分詞〉（過去への推量・後悔）

**本章の最重要項目です。**
通常の助動詞（can, must, may）は「現在」の推量を表しますが、後ろを `have + p.p.`（完了形）にすると、**「過去のこと」に対する推量や後悔**になります。

### 推量の対比表

| 助動詞 | 現在のことへの推量 ($V$原形) | 過去のことへの推量 ($have + p.p.$) |
| :--- | :--- | :--- |
| **must** | **must be** ...<br>（今）～にちがいない | **must have been** ...<br>（あの時）～**だったにちがいない** |
| **may** | **may be** ...<br>（今）～かもしれない | **may have been** ...<br>（あの時）～**だったかもしれない** |
| **cannot** | **cannot be** ...<br>（今）～のはずがない | **cannot have been** ...<br>（あの時）～**だったはずがない** |
| **should** | **should be** ...<br>（今）～のはずだ | **should have been** ...<br>（あの時）～**だったはずだ** |

### 後悔・非難の表現
「推量」ではなく、過去の事実に対する「気持ち」を表すパターンです。

- **should have + p.p.**
  - 「（実際はしなかったが）**～すべきだったのに**」
  - 過去への後悔や、相手への非難を表します。
- **need not have + p.p.**
  - 「（実際はしてしまったが）**～する必要はなかったのに**」

## 6. 助動詞を含む慣用表現

会話や長文で頻出の表現です。丸暗記しましょう。

| 表現 | 意味 | 備考 |
| :--- | :--- | :--- |
| **had better V** | **～しなさい** | ※「～した方がいい」というより**「命令・警告」**に近い強い表現。目上の人には使わない。 |
| **would rather V ... (than ~)** | （～するより）**むしろVしたい** | 控えめな希望。 |
| **cannot help V-ing** | **～せずにはいられない** | $= \text{cannot but V}$ |
| **have only to V** | **～しさえすればよい** | "All you have to do is V" と書き換え可。 |
| **dare V** | **あえて～する** | 否定文・疑問文で使われることが多い。<br> $How \ dare \ you ...!$ （よくもまあ...！） |
| **may well V** | **～するのももっともだ／おそらく～だろう** | 理由・根拠があって「当然～だ」という推量。 |
| **might as well V** | **～した方がよい／～するのも悪くない** | 消極的な提案。「他に良い選択肢もないし」というニュアンス。 |

> **まとめ**
> 助動詞は「話し手の心」です。
> - 「絶対そうだ！」と思えば **must**
> - 「ありえるかも」と思えば **may**
> - 「過去のことだけど、きっとこうだった」と思えば **must have p.p.**
>
> この「心の度合い」と「時制のズレ（have p.p.）」を意識して使い分けましょう。
