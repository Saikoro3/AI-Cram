# 第2章 文の種類

前章（第1章）では、英文を構成する「文の要素（S, V, O, C, M）」と、それらが作る「5つの文型」について学びました。
本章では、それらの文型を使って、どのような「種類」の文（肯定文、疑問文、命令文、感嘆文）を作ることができるのかを学習します。

特に、**「語順」の変化**や、**文頭に来る語（疑問詞や感嘆詞）が文中でどの「要素」の役割をしているか**という視点は、第1章の知識が土台となります。

---

## A. 平叙文・疑問文

### 1. 平叙文 (Declarative Sentences)
事実や意見をありのままに述べる、最も基本的な文です。
*   **肯定文**: 「～である」「～する」
*   **否定文**: 「～でない」「～しない」
    *   否定語 **not** の位置に注意しましょう（be動詞・助動詞の後ろ、一般動詞の前など）。

### 2. 疑問文 (Interrogative Sentences)

疑問文は大きく分けて以下の種類があります。

#### (1) Yes/No 疑問文
「はい」か「いいえ」で答えられる疑問文です。
文頭に **助動詞** (Do, Does, Did, Can, Will など) や **be動詞** を置きます。

#### (2) WH 疑問文 (疑問詞を用いた疑問文)
具体的な情報を尋ねる疑問文です。**Yes/No で答えることはできません。**
疑問詞（Who, What, Which, When, Where, Why, How）を文頭に置きます。

> [!IMPORTANT]
> **重要：疑問詞の「文の要素」としての役割**
> 第1章で学んだ S, O, C, M の視点で疑問詞を捉えることが、正確な理解への近道です。
> 疑問詞は単に文頭にあるだけでなく、**文の中で欠けている要素の代わり**をしています。

| 疑問詞 | 文中での役割の例 | 例文 (太字が疑問詞の役割) |
| :--- | :--- | :--- |
| **Who** | **S (主語)** | **Who** plays the piano? (誰がピアノを弾きますか？) |
| **What** | **O (目的語)** | **What** did you buy? (あなたは何を買いましたか？) <br> ※ *You bought [What].* の語順変化 |
| **When** | **M (修飾語)** | **When** will he come? (彼はいつ来ますか？) |

#### (3) 付加疑問文 (Tag Questions)
平叙文の後に短い疑問形を付け加え、「～ですよね？」と同意や確認を求める表現です。

> [!TIP]
> **付加疑問文のルール**
> *   前の文が **肯定** なら、後ろは **否定**
> *   前の文が **否定** なら、後ろは **肯定**

| 前の文 | 付加疑問 | 意味 |
| :--- | :--- | :--- |
| **肯定文** ($S + V...$) | ..., **否定形** ($don't / isn't...$) + $S$? | ～ですよね？ |
| **否定文** ($S + V + not...$) | ..., **肯定形** ($do / is...$) + $S$? | ～ではないですよね？ |

*   **You are a student, aren't you?** (あなたは学生ですよね？)
*   **He doesn't like natto, does he?** (彼は納豆が好きではないですよね？)

#### (4) 修辞疑問文 (Rhetorical Questions)
形は疑問文ですが、実際には答えを求めておらず、**強い肯定や否定（反語）**を表す表現です。

*   **Who knows?**
    *   直訳：誰が知っているだろうか？（いや、誰も知らない）
    *   意味：**誰も知らない。**

## B. 命令文・感嘆文

### 1. 命令文 (Imperative Sentences)
相手に対して「～しなさい」と命令・要求・依頼する文です。

> [!NOTE]
> **命令文の特徴**
> *   主語 **You** を省略する。
> *   **動詞の原形** で文を始める。

*   **肯定命令**: **Be** careful. (気をつけなさい。)
*   **否定命令**: **Don't** touch it. (それに触るな。)
*   **勧誘**: **Let's** go. (行きましょう。)

#### 命令文 + 接続詞 (and / or)
命令文の後に `and` や `or` を続けると、条件のような意味になります。

| 構文 | 意味 | 覚え方 |
| :--- | :--- | :--- |
| **命令文, and ...** | ～しなさい、**そうすれば**... | 順接（プラスの流れ） |
| **命令文, or ...** | ～しなさい、**さもないと**... | 逆接（マイナスの警告） |

*   **Study hard, and you will pass the exam.** (一生懸命勉強しなさい、そうすれば合格しますよ。)
*   **Hurry up, or you will be late.** (急ぎなさい、さもないと遅刻しますよ。)

---

### 2. 感嘆文 (Exclamatory Sentences)
「なんて～なんだろう！」と驚きや感動を表す文です。
**What** を使う形と **How** を使う形の2種類があり、この使い分けが非常に重要です。

> [!IMPORTANT]
> **What と How の決定的な違い**
> *   **What** は **名詞** と仲良しです。（名詞を強調する）
> *   **How** は **形容詞・副詞** と仲良しです。（形容詞・副詞を強調する）

#### (1) What を使う感嘆文
強調したい部分に **「名詞」** が含まれる場合に使います。

> **What + (a/an) + 形容詞 + 名詞 + S + V !**

*   元文: This is a very **cute cat**.
*   感嘆文: **What a cute cat** this is! (これはなんてかわいい猫なんでしょう！)

#### (2) How を使う感嘆文
強調したい部分が **「形容詞」または「副詞」だけ** の場合に使います。

> **How + 形容詞 / 副詞 + S + V !**

*   元文: This flower is very **beautiful**.
*   感嘆文: **How beautiful** this flower is! (この花はなんて美しいのでしょう！)

#### 構造的な理解（第1章との関連）
なぜ `What` と `How` で使い分けるのでしょうか？
第1章で学んだ「品詞」の役割を思い出してください。

*   疑問文の `What` は「何」という **代名詞（名詞の仲間）** でした。だから感嘆文でも **名詞** のカタマリ（a cute cat）を引っ張ってきます。
*   疑問文の `How` は「どのように」という **副詞** でした。だから感嘆文でも **形容詞や副詞**（beautiful）を直接引っ張ってくるのです。

| 種類 | 強調する対象 | 語順のポイント |
| :--- | :--- | :--- |
| **What** 感嘆文 | **名詞** (句) | What のすぐ後ろに **(a/an) + 形容詞 + 名詞** が来る |
| **How** 感嘆文 | **形容詞 / 副詞** | How のすぐ後ろに **形容詞 / 副詞** が来る |

---
**学習のポイントまとめ**
1.  疑問詞は文中で S, O, C, M のどの役割をしているか常に意識する。
2.  付加疑問文は「肯定⇔否定」のスイッチを忘れない。
3.  感嘆文は「名詞があるなら What」「形容詞・副詞だけなら How」と見極める。
