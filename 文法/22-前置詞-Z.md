# 第22章 前置詞

前回の第21章では、名詞を修飾する「形容詞」と、名詞以外を修飾する「副詞」について学びました。
今回の**第22章「前置詞」**は、名詞の前に置かれ、その名詞とセットになって「形容詞句」や「副詞句」を作る重要な品詞です。

前置詞は日本語の「～に」「～で」「～へ」などの助詞に相当しますが、1対1で対応するわけではありません。丸暗記ではなく、それぞれの前置詞が持つ**「コア・イメージ（中心的な意味の広がり）」**を理解することが習得の近道です。

---

## A. 前置詞の用法（コア・イメージ）

前置詞（Preposition）は、文字通り「前（Pre）に置く（Position）」言葉です。必ず**名詞（または名詞相当語句）**の前に置かれます。

ここでは、最も基本的かつ重要な3つの前置詞 **$at, on, in$** について、「時」と「場所」の両面からコア・イメージを整理します。

### 1. 時を表す $at, on, in$

| 前置詞 | コア・イメージ | 説明 | 例文 |
| :--- | :--- | :--- | :--- |
| **$at$** | **点** | 時計の針が指すような**「一点（時刻）」**を表します。 | $at \ 5 \ p.m.$ （午後5時に）<br>$at \ noon$ （正午に） |
| **$on$** | **接触・面** | カレンダー上の特定の日付や曜日など、**「特定の日」**を表します。 | $on \ Monday$ （月曜日に）<br>$on \ my \ birthday$ （誕生日に） |
| **$in$** | **内部・枠** | ある程度の長さを持つ**「期間・枠」**（月、年、季節など）を表します。 | $in \ May$ （5月に）<br>$in \ 2024$ （2024年に）<br>$in \ the \ morning$ （午前中に） |

> [!TIP]
> **イメージの広がり**
> *   $at$: ピンポイントな「点」。狭い時間。
> *   $on$: カレンダーに指を置くイメージ（接触）。1日単位。
> *   $in$: 枠の中に入っているイメージ。1日より長い期間や、午前・午後という枠。

### 2. 場所を表す $at, on, in$

「時」と同じイメージが「場所」にも適用されます。

| 前置詞 | コア・イメージ | 説明 | 例文 |
| :--- | :--- | :--- | :--- |
| **$at$** | **点** | 地図上の**「地点」**としての場所。狭い場所や、通過点・目的地としての場所。 | $at \ the \ station$ （駅で）<br>$at \ the \ door$ （ドアのところで） |
| **$on$** | **接触・面** | **「接触」**している状態。上だけでなく、壁や天井への接触も含みます。 | $on \ the \ desk$ （机の上に）<br>$on \ the \ wall$ （壁に） |
| **$in$** | **内部・空間** | 立体的な**「空間の中」**や、境界線で囲まれた**「広い場所」**（都市、国など）。 | $in \ the \ box$ （箱の中に）<br>$in \ Tokyo$ （東京で） |

## B. さまざまな前置詞

ここでは、間違いやすい前置詞のペアや、重要な前置詞を機能別に整理します。

### 1. 期限・期間・継続 ($by, until, for, during$)

| 前置詞 | 意味 | 使い分けのポイント | 例文 |
| :--- | :--- | :--- | :--- |
| **$by$** | ～までに | **「期限」**。その時までに動作が完了していればよい（完了の期限）。 | Finish it **by** tomorrow.<br>（明日までに終えなさい） |
| **$until / till$** | ～までずっと | **「継続」**。その時まで動作がずっと続いている（継続の終了点）。 | I waited **until** noon.<br>（正午までずっと待った） |
| **$for$** | ～の間 | **「数値的な期間」**。具体的な数字（three daysなど）が続くことが多い。 | I stayed there **for** three days.<br>（3日間滞在した） |
| **$during$** | ～の間 | **「特定の期間」**。特定の出来事や期間の名称（the vacationなど）が続く。 | I slept **during** the class.<br>（授業中寝ていた） |

> [!WARNING]
> **$by$ と $until$ の違い**
> 日本語ではどちらも「～まで」と訳せますが、意味は全く異なります。
> *   「提出期限は金曜**まで**」→ $by$ Friday （金曜以前に出してもOK）
> *   「金曜**まで**ここにいて」→ $until$ Friday （金曜までずっと居続ける）

### 2. 起点・手段・同伴 ($since, by, with$)

*   **$since$** （～以来）：
    *   過去の特定の時点を**「起点」**として、現在まで続いていることを表します。
    *   第4章「完了形」で学んだように、**現在完了形**とセットで使われます。
    *   例：$I have lived here **since** 2010.$ （2010年からずっと住んでいる）

*   **$by$** （～によって）：
    *   **「手段・方法」**を表します。交通手段や通信手段など。
    *   例：$go **by** bus$ （バスで行く）、$send **by** email$ （メールで送る）
    *   ※交通手段の場合、冠詞（a/the）を付けないことに注意（$\times by \ a \ bus$）。

*   **$with$** （～と一緒に、～を使って）：
    *   **「同伴」**：$go **with** him$ （彼と一緒に行く）
    *   **「道具」**：$cut **with** a knife$ （ナイフで切る）
    *   **「付帯状況」**：$stand **with** his eyes closed$ （目を閉じたまま立つ ※第10章・分詞構文に関連）

### 3. 位置関係 ($over, above, below, under$)

*   **$over$** （～の真上に、～を覆って）：
    *   何かに覆いかぶさるような「円弧」のイメージ。接触していても離れていてもOK。
    *   例：$fly **over** the mountain$ （山を越えて飛ぶ）
*   **$above$** （～より高い位置に）：
    *   基準よりも「レベルが上」。真上でなくてもよい。
    *   例：$The sun is **above** the horizon.$ （太陽が地平線より上にある）
*   **$under$** （～の真下に）：
    *   $over$ の反意語。何かの真下、影響下にあるイメージ。
    *   例：$under the table$ （テーブルの下）
*   **$below$** （～より低い位置に）：
    *   $above$ の反意語。基準よりも「レベルが下」。
    *   例：$below average$ （平均以下）

### 4. 移動・通過 ($through, across$)

*   **$through$** （～を通り抜けて）：
    *   トンネルのような立体的な空間を「貫通」するイメージ。
    *   例：$go **through** the tunnel$ （トンネルを通り抜ける）
*   **$across$** （～を横切って）：
    *   平面を「横断」するイメージ。
    *   例：$walk **across** the street$ （通りを横切る）

## C. 二重前置詞・群前置詞

### 1. 二重前置詞
前置詞が2つ重なって使われることがあります。
*   **$from behind...$** （～の後ろから）
    *   例：$A cat jumped out **from behind** the door.$ （猫がドアの後ろから飛び出した）

### 2. 群前置詞
2語以上がセットになって、1つの前置詞のような働きをするものを「群前置詞」と呼びます。長文読解でも頻出の重要表現です。

| 表現 | 意味 | 例文 |
| :--- | :--- | :--- |
| **$according to...$** | ～によれば | **According to** the news, it will rain.<br>（ニュースによれば、雨が降るだろう） |
| **$because of...$** | ～のために（理由） | The game was canceled **because of** rain.<br>（雨のために試合は中止になった） |
| **$in spite of...$** | ～にもかかわらず | He went out **in spite of** the cold.<br>（寒さにもかかわらず彼は外出した） |
| **$instead of...$** | ～の代わりに | I'll have tea **instead of** coffee.<br>（コーヒーの代わりに紅茶を頂きます） |

## D. 成句をつくる前置詞（コロケーション）

特定の動詞、形容詞、名詞は、特定の前置詞と強く結びつきます。これを**コロケーション（語の結びつき）**と呼びます。理屈よりも「セット」で覚えることが重要です。

### 1. 形容詞 + 前置詞
*   **$be different *from*...$**: ～と異なっている
*   **$be good *at*...$**: ～が得意である（一点（at）に集中して能力が高いイメージ）
*   **$be interested *in*...$**: ～に興味がある（対象の中（in）に入り込むイメージ）
*   **$be afraid *of*...$**: ～を恐れる

### 2. 動詞 + 前置詞
*   **$consist *of*...$**: ～から成る（構成されている）
*   **$depend *on*...$**: ～に頼る、～次第である（on = 接触・依存のイメージ）
*   **$look *for*...$**: ～を探す（for = 求める方向）
*   **$look *at*...$**: ～を見る（at = 視線の点）
*   **$wait *for*...$**: ～を待つ

---

## まとめ
前置詞は、名詞の前に置かれて、時や場所、状態などの情報を付け加える接着剤のような役割を果たします。
*   **$at, on, in$** のコア・イメージ（点・面・空間）を常に意識しましょう。
*   **$by$**（期限）と **$until$**（継続）のような機能的な違いを区別しましょう。
*   群前置詞やコロケーションは、一つの単語のようにセットで定着させましょう。
