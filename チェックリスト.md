# 高校英語カリキュラム チェックリスト

## 序論：目的と構成

- **目的**: 高校英語の全22章カリキュラムに沿って、指導の核心と実践的な留意点を章・節ごとに整理する。
- **利用手順**:
  1. 必要な章（時制・関係詞・仮定法など）を特定する。
  2. 章内の **節名(A/B/C...)** をキーにして、該当項目を参照する。
  3. さらに詳しい例文が必要な場合は元資料を参照し、ここでは構造と要点のみを即座に取り込む。

---

## 第1章 文の要素

### A 文の要素

- 文を構成する5要素: **S(主語)/V(動詞)/O(目的語)/C(補語)/M(修飾語)**。
- 品詞と文の要素は別レイヤーであり、名詞= S/O/C/前置詞のO、形容詞= Cまたは名詞修飾のM、副詞= Mのみとなる対応を明確化。
- この対応関係が文型理解や後続章（受動態・関係詞など）の土台になる。

### B 第1〜第3文型

- 第1文型 **S+V**: 自動詞中心で存在・進行を表す。しばしばMを伴う。
- 第2文型 **S+V+C**: S=C の関係。Cには名詞・形容詞のみ。
- 第3文型 **S+V+O**: 他動詞による S≠O の関係。
- 自動詞/他動詞の区別が受動態・関係代名詞・文型理解の核心。be動詞は第1/第2文型のみ。

### C 第4・第5文型

- 第4文型 **S+V+O₁+O₂**: 授与構文。to型(客体必須) と for型(客体任意) の区別が受動態の非対称性に直結。
- 第5文型 **S+V+O+C**: O=C の関係。Cには名詞・形容詞・不定詞・分詞など多様な形が来る。
- 使役・知覚動詞の構文理解につながる。

---

## 第2章 文の種類

### A 平叙文・疑問文

- 平叙文: 肯定/否定の基本構造と **not** の位置。
- Yes/No疑問文: 助動詞前置。WH疑問文は疑問詞が文中でどの文要素かを分析させる。
- 付加疑問文は主節が肯定なら否定、否定なら肯定のタグを付ける。修辞疑問文は反語的用法。

### B 命令文・感嘆文

- 命令文: You省略 + 動詞原形開始。否定は **Don't + V**。
- 感嘆文: `What + 名詞` と `How + 形容詞・副詞` の構造差を押さえる。
- 「... , and/or ...」構文で結果・警告を表す形も確認する。

---

## 第3章 基本時制

### A 現在時制

- 状態・習慣・不変の真理に加え、**確定的未来** や 時/条件節内の未来代用としても使われる。

### B 過去時制

- 「過去の一点」かつ「現在との断絶」がコア。過去完了との対比を常に意識させる。
- 礼儀的・仮定法的な距離感を表す用法にも触れる。

### C 未来を表す表現

- **will**: その場の意志 / 単純予測。
- **be going to**: 計画済みの意図 / 兆候に基づく未来。
- 現在進行形・be to・be about to などの未来表現とのニュアンス差を整理。

### D 進行形

- 「一時性・途中性」を表す。状態動詞は原則進行形不可だが、例外的に意味変化するケースも扱う。

---

## 第4章 完了形

### A 現在完了・現在完了進行形

- 過去形=「点・断絶」 vs 現在完了=「過去〜現在への幅・接続」。
- 完了/結果・経験・継続の3用法と、動作/状態の違い（**have been -ing** / **have been**）。

### B 過去完了・過去完了進行形

- 「過去の基準時」より前の大過去を示す。

### C 未来完了・未来完了進行形

- 「未来の基準時点までに〜しているだろう」という予測を表す。

---

## 第5章 助動詞

### A 助動詞の種類と性質

- 形態的ルール: 主語で変化なし・後ろは原形・否定/疑問での位置など。

### B can, may, must

- 多義性を文脈とともに整理。**must not** と **don't have to** の意味差を強調。

### C would, should

- will/shall の過去形としての用法 + 独自用法（過去の習慣・丁寧な依頼・仮定法など）。

### D その他の助動詞

- **ought to**, **used to** などの扱い。

### E 助動詞 + have + p.p.

- 現在から見た過去の推量・判断を示し、現在形との対照を示す。

### F 助動詞を含む慣用表現

- **had better**, **would rather**, **cannot help ...ing**, **have only to**, **dare**(助動詞) など。

---

## 第6章 受動態

### A 能動態と受動態、受動態の時制

- **SVO → O + be + p.p. (+ by S)** の変換ルール。他動詞のみが受動態をつくれる点を再確認。

### B 文型別の受動態

- **SVOO**: to型はO₁/O₂とも主語化可、for型は原則O₂のみ。
- **SVOC**: 使役・知覚動詞で **be made to do / be seen doing** など形の変化を把握。

### C 群動詞の受動態

- **look after / laugh at** など前置詞を含めたまとまりで受動態化。

### D 受動態の慣用表現

- **be interested in / be known to / be covered with** など、by 以外の前置詞をとる表現。

### E 受動態を作る一般動詞

- 自動詞(die, arriveなど)は受動態不可である点を再度確認。

---

## 第7章 不定詞

### A 名詞的用法

- **to V** が「〜すること」として S/O/C を担う。形式主語・形式目的語構文と連動。

### B 形容詞的用法

- 名詞 + **to V** の後置修飾。O関係/S関係/前置詞のO関係を整理。

### C 副詞的用法

- 目的・結果・原因/理由・判断根拠・形容詞の限定など多義。文脈判断を徹底。

### D 意味上の主語・否定・代不定詞

- **for/of + 名詞**（意味上の主語）、**not to V**、to のみを用いた省略表現。

### E 疑問詞 + to 不定詞

- **what/how/where to V** など、名詞句として S/O/C を担う。

### F 不定詞を含む慣用表現

- **too ... to / ... enough to / in order to / so as to** など。

### G 不定詞の完了形・受動態

- **to have + p.p. / to be + p.p.** の時制・態の扱い。

### H 原形不定詞

- 使役・知覚動詞 + O + 原形構文と、O + doing との意味差。

---

## 第8章 動名詞

### A 動名詞と意味上の主語

- 名詞的用法として S/O/C・前置詞のOになれる。意味上の主語は所有格/目的格で表す。

### B 否定形・完了形・受動態

- **not + V-ing / having + p.p. / being + p.p.** の時制・態の扱い。

### C 動名詞 vs 不定詞

- 不定詞=未実現・未来志向、動名詞=既実現・反復・一般的というコアイメージで使い分け。
- 不定詞のみ・動名詞のみ・両方で意味変化する動詞群を整理。

### D 動名詞の慣用表現

- **look forward to / be used to / There is no ...ing / feel like ...ing / on ...ing** など。

---

## 第9章 分詞

### A 限定用法

- 現在分詞=能動・進行「〜している」、過去分詞=受動・完了「〜された」。長い分詞句は名詞の後ろから修飾。

### B 叙述用法（主格補語）

- 感情動詞の分詞形 (**excited/exciting** など) を主語が「与える側/受ける側」で区別。

### C 叙述用法（目的格補語）

- 知覚・使役など第5文型でOの状態を説明 (**see O doing / have O p.p.** など)。

### D 分詞の慣用表現

- **boiling water / fallen leaves / developed countries** など。

---

## 第10章 分詞構文

### A 用法

- 接続詞+主語+動詞の副詞節を **V-ing** に圧縮。時・理由・譲歩・条件・付帯状況・結果の6解釈は文脈判断。

### B 位置形

- 文頭/文中/文尾での配置と、否定(**Not V-ing**)、完了(**Having p.p.**)、受動(**(Being) p.p.**)の形を押さえる。

### C 独立分詞構文

- 主節と異なる主語が明示された **S' + V-ing** の形。

### D 慣用的独立分詞構文

- **Generally speaking / Speaking of ... / Judging from ...** などの定型表現。

---

## 第11章 関係代名詞

### A 制限用法

- who/which/that の格(主格・所有格・目的格)と使い分け。関係詞節は関係代名詞位置に対応する名詞が欠けた「不完全文」になる点が重要。

### B 前置詞 + 関係代名詞

- 前置詞を節末に残す形 vs **前置詞+関係代名詞** の形。その際 **that不可・whomのみ** などの制約に注意。

### C 関係代名詞の省略

- 目的格のみ省略可能。

### D 非制限用法

- コンマ付きで追加情報を与える用法。**that不可** で、**which** が文全体を受ける用法も扱う。

### E その他の関係代名詞 (what)

- 先行詞を含む関係詞として **the thing(s) which** 相当の名詞節を作る。that節との違いを整理。

---

## 第12章 関係副詞・複合関係詞

### A 関係副詞

- **where/when/why/how** が副詞(M)として働き、後ろは完全文になる。
- **the reason why / the way (how不可)** など典型的な制限・非制限用法を整理。

### B 複合関係詞

- **whatever / whoever / whichever** など、先行詞不要で名詞節または譲歩の副詞節を導く。

---

## 第13章 接続詞・間接疑問文

### A 等位接続詞

- **and/but/or/so/for** が文法的・意味的に対等な要素を結ぶ。

### B 名詞節・形容詞節・副詞節を導く語

- **that/if/whether/when/where/why/how/what/which/who** などがどのタイプの節を導くか一覧で整理。

### C 主な接続詞の用法

- 名詞節（間接疑問・内容節）と副詞節（時・理由・条件・譲歩）での用法を例とともに明確化。
- 時・条件の副詞節では未来の内容でも現在形を用いる点を強調。

---

## 第14章 比較

### A 比較変化

- 原級/比較級/最上級の形の作り方と不規則変化。

### B 原級比較

- **as ... as / not as(so) ... as / as many/much ... as** など。

### C 比較級の表現

- 比較級 + than、強調語(**much/far/a lot**)、 **the 比較級..., the 比較級...** 構文。
- **more A than B / no less than / no more than / superior to** など特殊表現。

### D 最上級の表現

- **in/of** の使い分け、**one of the + 最上級 + 複数名詞**、序数 + 最上級。

### E 書きかえ

- 最上級 ⇔ 比較級 ⇔ 原級 (**no other / any other** の使い方含む) を相互変換できるよう整理。

---

## 第15章 仮定法

### A 仮定法過去

- 現在の事実に反する仮定を **If + 過去形, S + would/could/might + 原形** で表現。「時制のズレ」が非現実性を示す。

### B 仮定法過去完了

- 過去の事実に反する仮定 (**If + had + p.p., S + would/could/might have + p.p.**)。

### C were to / should

- 未来の非現実的・可能性の低い仮定(**were to**) と「万一〜なら」(**should**)。

### D 倒置による if の省略

- **Were I you / Had I known ... / Should you need ...** など。

### E if節以外の仮定表現

- **Without/But for + 名詞**, 主語構文, **To V, ... would ...** など。

### F 仮定法を含む慣用表現

- **I wish / as if(as though) / It's (high/about) time ...** などと仮定内容の時制対応。

### G 仮定法現在

- 要求・提案・命令を表す動詞 + that節内で **(should) + 原形** を用いる形。

---

## 第16章 話法の転換

### A 時制の一致

- **said/thought** など過去の主節に引きずられ、従属節の時制が1ステップ過去側にずれる原則と例外（不変の真理など）。

### B 平叙文の転換

- 直接話法→間接話法での人称・時制・時表現(**now→then** など)の変化。

### C 疑問文・命令文・感嘆文の転換

- **ask/tell** を用いた間接疑問、命令文→不定詞、感嘆文→that節/間接疑問への変換。

---

## 第17章 否定表現

### A 文全体の否定と語句の否定

- **not** による通常否定と、**no/never/nothing/nobody** など否定語が文成分になる場合の違いを区別。

### B さまざまな否定

- 部分否定(**not always / not necessarily** など)、全体否定(**none/neither** など)、二重否定(**not unusual, not ... without ...**)の意味を整理。

---

## 第18章 特別な表現

### A 強調

- **do/does/did** による動詞強調、**It is ... that ...** 構文による S/O/M の強調。

### B 倒置

- 疑問文型、否定語句の前置(**Never/Little/Not only**)、仮定法のif省略、**So ... that** の So 前置、V+S型倒置など。

### C 同格・省略・挿入

- 名詞同格構文(**the city of Tokyo** / 同格のthat節等)、動詞句の省略、挿入句の扱い。

### D 名詞構文・無生物主語

- 動詞的内容を名詞で表す硬い文体と、無生物主語構文を「副詞的に訳す」指導。

---

## 第19章 名詞・冠詞

### A 名詞の種類

- 可算/不可算を区別し、代表的不可算名詞(**furniture, information, advice** 等)を提示。

### B 名詞の格

- 所有格の原則: 人・生物は **A's B**、無生物は **B of A**。

### C 不定冠詞 a/an

- 「不特定」かつ「単数」というコアイメージ。可算名詞の単数形にのみ付く。

### D 定冠詞 the

- 「特定」されているものに付く。可算/不可算・単複を問わず使用可能。
- 既出・文脈・唯一無二・最上級・**the + 形容詞(〜な人々)** などの用法を整理。

### E 無冠詞

- 食事・スポーツ・建物の本来用途など、冠詞を省く典型パターンを押さえる。

---

## 第20章 代名詞

### A 人称・所有・再帰代名詞・it の特別用法

- 人称代名詞の格(主格/所有格/目的格)、所有代名詞、再帰代名詞の2用法(本来/強調)。
- **it**: 天候/時間/距離/状況/形式主語・目的語など多義。

### B 指示代名詞・疑問代名詞

- **this/that/these/those** と **that(those) of ...** による反復回避。
- 疑問代名詞 **who/whose/whom/what/which** の S/O/C としての機能。

### C 不定代名詞

- **one/another/the other/the others**, **each/every**, **both/either/neither**, **all/some/any/none** などの区別を明確化。

---

## 第21章 形容詞・副詞

### A 形容詞の種類と用法

- 限定用法 vs 叙述用法、叙述用法のみの形容詞(**afraid, asleep** など)。
- **something/anything + 形容詞** の語順を確認。

### B 数量を表す形容詞・数詞

- 可算(**many/(a) few**) vs 不可算(**much/(a) little**)、**a** の有無によるニュアンス差を整理。

### C 副詞の種類と用法

- 名詞以外を修飾するMとしての機能と、**hard/hardly, late/lately, near/nearly** など紛らわしいペア。
- **enough** の位置（形容詞・副詞の後ろ / 名詞の前）。

---

## 第22章 前置詞

### A 前置詞の基本用法

- 時: **at/on/in** のコアイメージ (時刻/日付/期間)。
- 場所: **at/on/in** のコアイメージ (地点/接触/内部)。

### B さまざまな前置詞

- **by/until/for/during/since/with/over/above/below/through/across** などの基本イメージ。

### C 二重前置詞・群前置詞

- **from behind, according to, because of, in spite of, instead of** など。

### D 成句をつくる前置詞

- **be different from / be good at / consist of / depend on / look for** など動詞・形容詞・名詞とのコロケーション。
